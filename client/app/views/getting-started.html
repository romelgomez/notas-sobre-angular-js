Estableciendo el entorno básico:

Antes de empezar primero vamos a disponer del entorno básico para crear una aplicación simple en AngularJS.

  1. Un Editor de texto.
  2. Un navegador web.
  3. Un servidor local o externo.
El siguiente código HTML5 sera nuestra plantilla base:

Directivas usadas: ng-app 

<!DOCTYPE html>
<html ng-app>
<head>
    <meta charset = "UTF-8" />
    <title>AngularJs - Plantilla Base</title>
    <link rel="stylesheet" href="css/bootstrap.min.css" media="screen">
</head>
<body style="padding:10px;border:1px dashed black;border-radius:3px;margin:10px;">

<h5>Plantilla Base</h5>

<script type="text/javascript" src="js/angular.min.js"></script>
</body>
</html>
Explicando la directiva:
  1. La directiva ng-app define el dominio donde AngularJS va a trabajar. En este caso es en todo el documento html. Vale destacar que sólo una directiva ng-app por documento HTML puede ser declarada.

Vale destacar que AngularJS esta integrado parcialmente con Bootstrap por lo tanto solo es necesario indicar la cascada y las imágenes para disponer de prácticamente todas las características o ventajas que tiene Bootstrap, además también está integrado con la versión liviana de jQuery llamada jqLite.

Descarga e instala Bootstrap, vamos a usarlo para crear vistas agradables a la vista. Link: http://twitter.github.com/bootstrap/index.html

Si la versión completa de jQuery es cargada antes que ocurra el evento DOMContentLoaded, tendrá prioridad sobre la versión integrada de jqLite.

La lista completa de los métodos de jqLite esta disponible en la sección: global APIs, bajo el nombre: angular.element, Link: http://docs.angularjs.org/api/angular.element
